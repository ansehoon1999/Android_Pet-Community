## Explanation

많은 사람들이 키우고 있지만 바쁜 현대인들에게 계속해서 반려견을 보살피는 일은 쉬운 일은 아닙니다. 따라서 주변 이웃과 서로 반려 동물을 맡길 수 있는 안드로이드 플랫폼을 만들었습니다.

## Function

- 손님과 주인 사이의 예약 매칭 및 채팅 시스템
- 손님과 주인 각각의 정보 입력 시스템
- 반려 동물 알러지 방지 OCR 기능
- 산책 및 상대 위치 GPS 기능
